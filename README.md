# 🔢 Arabic Numerals Classification using Deep Learning

End-to-end deep learning system for Arabic numerals classification using CNN, ResNet50, VGG16, and Vision Transformers (ViT), with full training, evaluation, and performance comparison.

---

## 🚀 Overview

This project focuses on building high-performance deep learning models to recognize Arabic numerals from images.
Multiple architectures are implemented and compared to achieve the best accuracy and generalization.

---

## 🧠 Models Used

* Convolutional Neural Network (CNN)
* ResNet50
* VGG16
* Vision Transformer (ViT)

---

## 📊 Evaluation Metrics

* Accuracy
* Precision, Recall, F1-score
* Confusion Matrix
* Training & Validation Curves

---

## 📂 Project Structure

```
notebooks/   # Training notebooks (CNN, ResNet, VGG16, ViT)
results/     # Output images (plots, confusion matrices)
reports/     # Classification reports and evaluation files
```

---

## 🛠️ Technologies

* Python
* TensorFlow / Keras
* NumPy & Pandas
* Matplotlib & Seaborn

---

## 📈 Results

The models were trained and evaluated on Arabic numerals datasets, achieving strong performance across multiple architectures.
Comparisons show differences in accuracy, speed, and generalization between CNN and transformer-based models.
